# shop-demo

Simple and clean online shoe shop demo showcasing a product gallery and basic e‑commerce UI experience. It highlights a range of shoe products with images, categories, and pricing, Showcasing a clean and responsive product gallery interface.

## Features

• Product display grid – Shows shoe items with images, names, and prices.

• Basic UI navigation icons – Includes cart, search, and chat icons at the top.

• Categories & filters (UI only) – Displays shoe types like Sneakers, Boots, Sandal, Heels, Loafers.

• Brand filters (UI only) – Shows popular brands like Nike, Adidas, Puma, Converse, Timberland.

• Mobile‑friendly layout – Simple responsive grid for browsing products on different screen sizes.

## Shop Shack

• HTML – Page structure

• CSS – Styling and layout

• Images and icons to represent products and UI elements
